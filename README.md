# Web3LearningApp
App educacional para ensinar Web3, HTML, CSS, JS e mais, com React Native e Firebase.

## Setup
- Instale dependências: `npm install`
- Inicie o projeto: `npx expo start`
- Teste no iPhone via Expo Go.
